BSON

BSON (Binary JSON) é um formato de serialização binária usado para armazenar documentos e arrays. É usado principalmente no MongoDB.

JSON

JSON (JavaScript Object Notation) é um formato de texto legível para humanos, usado para transmitir dados estruturados entre aplicações.

Esta ferramenta tenta converter qualquer arquivo para JSON. Arquivos não-BSON resultarão em JSON vazio ou erro controlado.
